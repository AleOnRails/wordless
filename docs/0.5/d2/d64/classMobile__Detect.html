Detailed Description

Mobile Detect Library

Motto: "Every business should have a mobile detection script to detect mobile readers"

Mobile_Detect is a lightweight PHP class for detecting mobile devices (including tablets). It uses the User-Agent string combined with specific HTTP headers to detect the mobile environment.

Author
Current authors: Serban Ghita serba.nosp@m.nghi.nosp@m.ta@gm.nosp@m.ail..nosp@m.com, Nick Ilyin nick..nosp@m.ilyi.nosp@m.n@gma.nosp@m.il.c.nosp@m.om Original author: Victor Stanciu vic.s.nosp@m.tanc.nosp@m.iu@gm.nosp@m.ail..nosp@m.com
